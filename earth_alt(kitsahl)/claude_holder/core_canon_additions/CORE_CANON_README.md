## Usage Guidelines

**Canonical Status**: Everything in Core Canon is **fixed**. Other sections (geography, culture, daily life) can be flexible, but Core Canon defines the unchangeable foundation.

**Cross-References**: When referencing Core Canon from other sections, use the format: `[See: Core Canon > Timeline > Oracle Wars]`

**Contradiction Resolution**: If any other worldbuilding contradicts Core Canon, Core Canon wins. Update the other content.

---

## Design Principles

1. **Grounded in History**: Use real industrial-era trends, real geography, real economic principles
2. **Alternate, Not Fantasy**: The only divergence is oracle technology; everything else follows logically
3. **Colonial Critique**: Every oracle device represents stolen indigenous knowledge
4. **Mechanical Spirituality**: Europeans stripped the spiritual meaning, kept the technical function
