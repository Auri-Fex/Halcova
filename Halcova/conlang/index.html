Conlang Language Set: World A

See also World Summary, Metaphysics, Magic Systems, Fate System, Society, Governance, and Planetary.

Selected Languages

Role Language Reasoning
Root Hungarian Rich vocabulary for abstract and visceral concepts. Agglutinative structure provides flexible roots. Unique among European languages—feels ancient and unfamiliar.
Affix Arabic Distinct phonetic character. Tri-consonantal roots allow for varied affix extraction. Carries inherent weight and formality. Religious/mystical connotations fit dark fantasy.
Mutation Scots Gaelic Soft mutations create flowing transformations. Celtic character adds otherworldly quality. Balances Hungarian's harshness with musicality.

Body/Physical Affixes

  • -lahm (flesh) → -laimh, -lav
  • -jism (body) → -gism, -gis
  • -jasad (corpse) → -jasad, -jas

Fate/Path Affixes (Fate System)

  • -qadr (destiny) → -cadr, -cad
  • -nasib (fortune) → -nasiv, -nav
  • -tariq (path) → -taric, -tar

Usage Notes

  1. Formal vs. Common: Longer unmolted forms for ritual/legal/religious contexts; molted forms for everyday speech.
  2. Compounds: Use shorter affix variants and compress aggressively.
  3. Names: Personal names use different affix sets for family, profession, or origin.
  4. Loanwords: Foreign magic system concepts retain more structure.

This document serves as the foundation for all vocabulary creation in World A.
