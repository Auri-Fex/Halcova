## Quick Reference Template

Each landing page should have:

1. **Header Section**
   - Category name
   - Tagline/description
   - Visual icon/color scheme

2. **Quick Facts Box**
   - 6-8 key statistics
   - Core defining elements
   - At-a-glance overview

3. **Browse Topics Section**
   - 3-4 major subsections
   - 3-6 links per subsection
   - Logical organization

4. **Related Categories**
   - Primary connections (3-4)
   - Secondary connections (3-4)
   - Cross-linking strategy

5. **Featured Content**
   - New visitor starting point
   - Popular/essential articles
   - Deep dive recommendations

6. **Navigation Footer**
   - Return to home
   - Browse all categories
   - Search function (if available)
