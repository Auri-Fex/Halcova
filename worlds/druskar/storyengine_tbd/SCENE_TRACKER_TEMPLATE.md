# DRUSKAR SAGA — Scene Tracker Template

## Scene Card Format

```
### SCENE [##]: [Title]
**Chapter:** [##]
**POV:** [Character name]
**Location:** [Place name]
**Timeline:** [LY##, LM#, W#, or relative: "Day 3", "Next morning"]

**Characters Present:**
- [Name] (role)
- [Name] (role)

**Scene Goal:** [What needs to happen for the plot]
**POV Goal:** [What the POV character wants]
**Conflict:** [What opposes the goal]

**Key Information:**
- [What the reader learns]
- [What the POV character learns]

**Evidence/Forensics:** [If applicable]
- [Evidence created]
- [Evidence discovered]
- [Evidence trail implications]

**System Elements:** [If applicable]
- [ ] Fortune: [Bends used? Claim made? Collection event? Drift shown?]
- [ ] Fortune Tactics: [Ornan counter-tactics in play?]
- [ ] Time: [Rig use? Costs shown?]
- [ ] Density: [Rig use? Strain shown?]
- [ ] Ossir: [Signal type? Error mode? Calibration?]
- [ ] Chronomass: [Present? Spoiling? Heat?]

**Fortune Tracking:** [If Fortune use occurs]
- Bend or Claim: [Which?]
- Clean or Dirty: [If claim, which type?]
- Residue created: [Signature left?]
- Collection pressure: [Rising? Settling?]
- Drift direction: [What pattern is accumulating?] 

**Emotional Beat:** [How does the POV character feel by scene end?]

**Hooks:**
- Sets up: [Future scene/plot point]
- Pays off: [Previous scene/plant]

**Word Count Target:** [Approximate]
**Status:** [ ] Outlined [ ] Drafted [ ] Revised [ ] Final
```

## Scene Type Reference

### Action Scenes
- Clear goal and obstacle
- Concrete stakes
- Equipment/environment matters
- Consequences persist

### Investigation Scenes
- Evidence discovered
- Deductions made
- Questions raised
- Timeline advanced

### Political Scenes
- Power dynamics shown
- Alliances shift
- Information traded
- Subtext matters

### Procedure Scenes
- System in action
- Bureaucracy as obstacle/tool
- Forms and slips
- Evidence creation

### Character Scenes
- Relationship development
- Internal change
- Quiet moments
- Emotional stakes

## Tracking Conventions

### Status Markers
- [ ] Not started
- [~] In progress
- [x] Complete
- [!] Needs revision
- [?] Uncertain/blocked

### Priority Markers
- **[P1]** Must happen
- **[P2]** Should happen
- **[P3]** Nice to have

### Connection Markers
- **→** Sets up
- **←** Pays off
- **↔** Parallels
