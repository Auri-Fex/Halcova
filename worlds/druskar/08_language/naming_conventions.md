# NAMING CONVENTIONS — People, Places, Devices

## Person Naming

### Three Registers

| Register | Format | Example |
|----------|--------|---------|
| **Stamped** | Given + Line + Mark | Tarek Verran Gate·14 |
| **Trade** | Given + Line | Tarek Verran |
| **Street** | Clipped Given/Line or nickname | Tarek / Tar / "Gate-Tarek" |

### Line Types
**Line** is one affiliation type—not stacked. Choose one:

| Type | Indicates | Example |
|------|-----------|---------|
| **Place** | Origin or residence | Tarek Knalvoss |
| **House** | Seal House or Belt House | Tarek Verran |
| **Works** | Employer or trade | Tarek Hullvek |
| **Guild** | Professional association | Tarek Burner |
| **Lineage** | Family name | Tarek Osten |

### Mark (Stamped Register Only)
Marks identify specific authorization or position:
- Gate·14 = Gate assignment 14
- Bench·3 = Seal bench assignment 3
- Order·Gaturline = Order Company affiliation

## Place Naming

### Settlement Pattern
[Root] + [Function Suffix]

| Suffix | Function |
|--------|----------|
| -ak | Commission seat |
| -am | Market hub |
| -vek | Works/yard |
| -esh | Port/quay |
| -ur | Fort |
| -keth | Checkpoint |
| -il | Lock-village |
| -ot | Outstation |

### Bare Names
Historic cores and infamous locations often have no suffix:
- Pechaten (old clerk row)
- Belkran (oldest charter stone)
- Lanterncut (posted narrow cut)
- Brinefield (low flats)

### Tract Names
Descriptor + bare name:
- Jitknal Labyrinth
- Kronvar Belt
- Brinav March
- Kestnar Step
- Varnek Flats

[Place/Territory] + [Function word]

## Device/Cargo Marking

### Dot-String Format
Marks tie: certification + permit class + authority routing

**Components:**
- Grade code (S = Set, R = Raw, C = Cut)
- House code (VER = Verran, KEL = Kelm, ORV = Orvek)
- Routing code (K = Knalvossak, KR = Krovam, RA = Raskesh)
- Lot/sequence number

**Example:** `S3·VER·K·14`
- Set grade 3
- Verran sealed
- Knalvossak routed
- Lot 14

### Counterfeit Tells
Counterfeiters usually misroute marks:
- Plate swaps (serial doesn't match wear)
- Partial restamps (ink depth variation)
- Forged permit class (routing mismatch)

## Document/Slip Codes

### Format
`CLASS·YEARCODE·SUBCLASS·SEQUENCE`

### Examples

| Code | Meaning |
|------|---------|
| `BA·26·2·14` | Bloom Advisory, year 26, Tier 2, sequence 14 |
| `CO·26·QC·7` | Closure Order, year 26, Quarantine Class, sequence 7 |
| `RN·26·31` | Reopen Notice, year 26, sequence 31 |
| `AR·VER·26·8` | Admissibility Review, House Verran, year 26, sequence 8 |
| `DR·GAT·26·44` | Detain Receipt, Gaturline Order, year 26, sequence 44 |
