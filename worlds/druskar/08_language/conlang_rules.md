# DRUSKAR CONLANG — Rules and Phonology (v0.3)

## Core Principles

1. **No hyphens** — Ever
2. **Compounds concatenate** — Words join directly
3. **Insert single 'e'** only to avoid ugly consonant clusters
4. **Dot-only formatting** — Middle dot (·) for official marks and codes

---

## Phonology

### Consonants
| Type | Examples |
|------|----------|
| Stops | p, b, t, d, k, g |
| Fricatives | f, v, s, z, sh |
| Nasals | m, n |
| Liquids | l, r |
| Glides | j (y-sound) |

### Vowels
| Type | Examples |
|------|----------|
| Short | a, e, i, o, u |
| Common patterns | -ak, -am, -esh, -il, -ur |

### Cluster Rules
- Avoid three consonants in sequence
- Insert 'e' as epenthetic vowel when needed
- Favor CV and CVC syllables

---

## Stress Patterns

### Simplex Words (Single Stem)
**Rule:** Stress falls on the **first syllable**
- KNALvoss
- PULjitos
- BRInav

### Compound Words
**Rule:** Stress falls on the **head stem** (rightmost meaningful stem before suffix)
- KnalVOSSak (Knalvoss + ak)
- GatuRline (Gatur + line)

---

## Three Registers

### 1. STAMPED (Formal/Written)
- Conservative spellings
- Full forms
- Used in: official documents, legal papers, formal address

**Example:** Knalvossak

### 2. TRADE (Professional/Commercial)
- Shorter forms
- Readable but practical
- Used in: business dealings, posted slips, casual professional contexts

**Example:** Knalvoss

### 3. STREET (Informal/Colloquial)
- Clipped forms
- Suffix-dropping when context allows
- Used in: daily speech, informal contexts, speed

**Example:** Voss

- Classification via **descriptor words** + **dot tract codes**

| Code | Name | Descriptor |
|------|------|------------|
| T1·Jitknal | Jitknal | Labyrinth |
| T2·Kronvar | Kronvar | Belt |

## Dot Formatting

**Rule:** Dots only (middle dot ·), no hyphens

### Tract Codes
Format: `T#·Name`
Example: `T1·Jitknal`

### Slip/Document Codes
Format: `CLASS·YEARCODE·SEQ`
Examples:
- `BA·26·2·14` (Bloom Advisory)
- `CO·26·QC·7` (Closure Order, Quarantine Class)
- `DR·Gaturline·26·31` (Detain Receipt)

### Device/Cargo Marks
Dot strings tie: certification + permit class + authority routing
Example: `S3·VER·K·14` (Set grade 3, Verran sealed, Knalvossak routed, lot 14)

## Compound Formation

### Basic Rule
Stems concatenate directly: Knalvoss + ak = Knalvossak

### Epenthetic 'e'
Insert when consonant cluster is ugly:
- Gatur + line = Gaturline (no e needed; acceptable cluster)
- [hypothetical] Brack + snail = Brackesnail (if cluster is harsh)

### Head Position
The meaningful stem is typically rightmost before the suffix:
- Gatur (fort) + line (company/order) = Gaturline
- Knalvoss (city name) + ak (commission seat) = Knalvossak

## Name Patterns

### Place Names
Usually: [Descriptive/Historic root] + [Function suffix]
- Krovam = [Krov-] + [-am] (belt market)
- Hullvek = [Hull-] + [-vek] (hull yard)

### Organization Names
Usually: [Place/Domain] + [Type word]
- Gaturline Order
- Krovam Shield Service
- Reedward Contract Patrol

### House Names
Single-word proper names:
- House Velt
- House Ornan
- House Kivren
- House Verran
- House Kelm
- House Orvek
