## Phase 2: Repetition Elimination

### "Something" Replacements (47 instances → target: <15)

**Pattern: "something + adjective"**

**Lines to revise** (Representative examples):

#### Instance 1: "Something complicated"
```
ORIGINAL: "Csèndmaht's face did something complicated—not quite confusion, not quite pain."

REVISED: "Csèndmaht's face shifted—not quite confusion, not quite pain. The muscles moved without comprehension behind them."
```

#### Instance 2: "Something that looked like..."
```
ORIGINAL: "her mother's face had worn something that looked like guilt"

REVISED: "her mother's face had worn guilt—unmistakable, quickly hidden"
```

#### Instance 3: Generic "something"
```
ORIGINAL: "Something exists that looks at human souls"

REVISED: "The Attendance exists. It looks at human souls"
```

### Strategy for All "Something" Instances:
1. **If describing appearance**: Name the specific expression/object
2. **If describing emotion**: Use the emotion word directly
3. **If describing action**: Describe the specific movement
4. **If describing abstraction**: Make concrete or restructure sentence
