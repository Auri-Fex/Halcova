# Selnara Conlang System

## Source Language (LOCKED)

**Base**: Georgian (ქართული)

**Mutation Method**: Georgian roots undergo systematic transformation to create unique Selnaran words while maintaining linguistic consistency and Georgian "flavor."

### Mutation Rules: Georgian → Selnaran

| Rule | Georgian | Selnaran | Example |
|------|----------|----------|---------|
| **Cluster simplification** | CC- → C- or -CVC- | Drop or insert vowel | *mdinare* → *dinar* |
| **Ejective softening** | k', t', p', ch' → k, t, p, sh | Plain stops | *t'ba* → *taba* |
| **ღ (gh) shift** | gh → g or v | Simplify | *zghva* → *zeva* |
| **ძ/წ (dz/ts) shift** | dz/ts → z/s | Simplify | *dzvel* → *zvel* |
| **ჭ (ch') shift** | ch' → sh or k | Avoid ejective | *ch'a* → *sha* |
| **Final syllable trim** | -i, -e endings | Often dropped | *kalaki* → *kalak* |
| **Vowel regularization** | Georgian vowels → i, e, a, o, u | 5-vowel fit | as needed |

## Phonology (LOCKED)

### Consonants

**Stops**: p, t, k, b, d, g

**Fricatives**: f, v, s, sh, h

**Nasals**: m, n

**Liquids**: l, r

**No clusters** at syllable start; maximum **one** at syllable end (typically /n/, /r/, /l/, /s/)

### Vowels

**i, e, a, o, u** (five-vowel system, no length distinction)

### Syllable Structure

**(C)V(C)**

* Open syllables (CV) preferred
* Closed syllables (CVC) allowed
* No consonant clusters at syllable start
* Maximum one consonant at syllable end

### Stress

**Penultimate stress** (second-to-last syllable)

* **ˈsel-na-ra** (not sel-na-ˈra)
* **kaˈlak-zeva** (not ˈka-lak-ze-va)

## Naming Rules (LOCKED)

### Place Names

**2–4 syllables**, penultimate stress

**Formation Patterns**:
* Root + Root: **Zeva-kalak** (sea-city)
* Root + Suffix: **Sharem** (pit-place)
* Modified Root: **Dinarel** (river-settlement)

**Common Place Suffixes**:
* **-kalak** — city, major settlement
* **-safek** — terrace, stepped settlement
* **-koshk** — tower, fortified place
* **-navsad** — port, harbor town
* **-el/-al** — place (diminutive)

### Person Names

**2–3 syllables**, simple structure

**Common Patterns**:
* **Root + -i/-is**: Sulari, Natelis
* **Root + -en/-an**: Kavaren, Benelen
* **Root + -el/-al**: Gozael, Metael
* **Compound**: Rikin-sul, Dinar-kav

### Faction/Institution Names

**Selnaran root + English descriptor** (loan-translation feel)

Examples:
* **Beghel Syndics** (seal cartels)
* **Khergem Brokers** (spine monopolists)
* **Navsad Arbiters** (pier networks)
* **Tavshes Enclaves** (refuge-ports)

### Creature Names (LOCKED)

**English descriptive + Selnara loanword feel**

Examples:
* **Ashwrights** (ash-creatures)
* **Varnish Sheets** (varnish-film-entities)
* **Pale-Gilled** (pale-contaminated-amphibians)
* **Signal Mimics** (signal-echo-beings)

## Usage Guidelines for Writers

### When to Use Selnaran Names

✅ **All geography**: cities, rivers, regions, landmarks

✅ **Major factions**: when creating new institutions

✅ **Named characters**: when cultural authenticity matters

✅ **Proper nouns**: ships, buildings, streets

### When to Use English

✅ **Common nouns**: street, river, gun, lamp

✅ **Technology terms**: telegraph, revolver, locomotive

✅ **Creature names**: Ashwright, Varnish Sheet (descriptive feel)

✅ **Institutional types**: Beghel Syndics, Navsad Arbiters (loan-translation feel)

### Avoiding Common Mistakes

❌ **Don't**: Use consonant clusters at syllable start (*Straven*, *Kralmor*)

✅ **Do**: Keep syllables simple (*Kavaren*, *Metakalak*)

❌ **Don't**: Use Earth-specific language (*Sheriff*, *Whitechapel*, *Fort Grey*)

✅ **Do**: Use Selnara equivalents (*Beghel Warden*, *Zevakalak docks*, *Kidkoshk*)

❌ **Don't**: Over-apostrophe (*K'thar'an*)

✅ **Do**: Keep it clean (*Ketharan*)

❌ **Don't**: Ignore Georgian derivation (*Selthar*, *Girven* — old system)

✅ **Do**: Use Georgian-derived roots (*Zevakoshk*, *Metakalak*)

## Quick Reference Generator

### Building a Place Name

1. Pick a geographic type: coast, pit, river, mountain, valley, edge
2. Find the root: zeva, sha, dinar, meta, keoba, kid
3. Add ending: -kalak, -safek, -koshk, -el, -navsad
4. Check stress: penultimate syllable

**Example**: Need a mountain fortress → **meta** (mountain) + **-koshk** (tower) = **Metakoshk** (stress: **me-ˈta-koshk**)

### Building a Person Name

1. Pick aesthetic: strong, subtle, occupational
2. Choose root: kava (stone), natel (light), goza (road), etc.
3. Add personal suffix: -en, -is, -rin, -el, -ari
4. Check syllable count: 2–3
5. Check stress: penultimate

**Example**: Need a light-associated character → **natel** (light) + **-is** (personal) = **Natelis** (stress: **na-ˈte-lis**)

### Deriving New Words from Georgian

1. Find Georgian word: ციხე *tsikhe* (fortress)
2. Apply mutation: ts→s, simplify = **sikhe** or **sike**
3. Check syllable structure: (C)V(C) ✓
4. Check phoneme inventory: s, i, k, e ✓
5. Result: **sike** — fortress, stronghold

**Example derivations**:
* თოვლი *tovli* (snow) → **tovli** or **toli**
* მზე *mze* (sun) → **meze** or **mez**
* მთვარე *mtvare* (moon) → **tavare** or **tavar**
