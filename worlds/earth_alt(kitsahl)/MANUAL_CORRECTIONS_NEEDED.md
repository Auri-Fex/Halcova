## üéØ Why This Can't Be Automated

`.gdoc` files are **not text files**‚Äîthey are shortcuts/links to Google Docs in your Google Drive. They contain only:

- A URL pointing to the Google Doc
- Metadata about the document

The actual document content lives in Google Drive and can only be edited through:
- Google Docs web interface
- Google Docs desktop app
- Google Drive API (requires authentication and special setup)

File system tools (like text editors or find/replace scripts) **cannot access** the actual document content.
